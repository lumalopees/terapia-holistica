# Serviços Holísticos — Landing Page

Uma landing page responsiva desenvolvida para um profissional de bem-estar e terapia holística. O projeto foca em traduzir requisitos de alto nível e vagamente definidos em uma interface de usuário clara, funcional e acessível.

## Visão Geral

Esta é uma aplicação de página única (landing page estilo SPA) projetada para estabelecer presença digital, organizar informações de serviços e guiar usuários em direção a uma ação principal (contato via WhatsApp). O layout prioriza design mobile-first, legibilidade e estrutura semântica, garantindo uma experiência suave em diferentes tamanhos de tela!

## Funcionalidades

- Layout totalmente responsivo (mobile-first)
- Estrutura de página única com hierarquia de conteúdo clara
- Layout modular baseado em seções
- Chamada para ação com redirecionamento para WhatsApp
- HTML semântico para acessibilidade e SEO
- Estrutura limpa e de fácil manutenção

## Escopo do Projeto

- Landing page de uma página
- Implementação de layout e estrutura de UI
- Hierarquia e organização de conteúdo
- Direção visual (cores, espaçamento, tipografia)
- Compatibilidade entre navegadores e dispositivos

**Nota:** Nenhuma identidade de marca ou cópia finalizada foi fornecida anteriormente. O projeto inclui estruturação básica de conteúdo e tomada de decisão visual para garantir consistência e clareza.
